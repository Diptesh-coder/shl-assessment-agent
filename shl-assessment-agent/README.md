# SHL Assessment Agent

## Setup

1) Create and activate a virtual environment.
2) Install dependencies:

```
pip install -r requirements.txt
```

3) Create a .env file:

```
cp .env.example .env
```

4) Add your Groq API key to .env:

```
GROQ_API_KEY="your_key_here"
```

## Run

Streamlit app:

```
streamlit run app.py
```
